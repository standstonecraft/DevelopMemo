# git-and-vscode

Windows の場合。

## SVNでいうチェックアウト的な行動

1. install git gui
1. launch vscode
1. <kbd>F1</kbd> `Git: Clone`
1. choose repository location (not repository folder)
1. click 'open' in notification balloon

どこかでgithubのログインページが開いたと思う

## push
