# npm よりも Yarn

##前提

[Node.js](https://nodejs.org/en/)

## 導入

管理者権限 required

```powershell
# PowerShell
choco install yarn
```

プロジェクト(フォルダ)で Yarn を有効にする

```powershell
# PowerShell
yarn init
```

## 更新

> Note: self-update is not available.

[yarn self-update | Yarn](https://classic.yarnpkg.com/en/docs/cli/self-update)

```bat
npm uninstall yarn -g
npm install yarn -g
```
